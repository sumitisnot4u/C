
# C語言的3個loop:for / while /do while


### 使用3種loop技術完成1+2+3+4+....+9+10

[1]for loop
```

#include <stdio.h>
#include <stdlib.h>
int main(void)
{
   int i,sum=0;
   
   for(i=1;i<=10;i++)			
      sum+=i;
      
   printf("1+2+3+...+10=%d\n",sum);	
   
   system("pause");
   return 0;
}
```
